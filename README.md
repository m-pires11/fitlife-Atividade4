# FitLife - SPA Interativo
[Veja o projeto online](https://m-pires11.github.io/fitlife-Atividade4/)

## Descrição
O **FitLife** é um site de treinos com páginas de **Início, Treinos e Cadastro**, totalmente funcional e responsivo.  
O formulário de cadastro possui **mensagem de sucesso ao enviar** e pode ser testado diretamente pelo navegador.

---

## Funcionalidades
- Navegação entre páginas Início, Treinos e Cadastro  
- Menu hambúrguer responsivo em telas pequenas  
- Formulário de cadastro funcional com mensagem de sucesso  
- Layout moderno e responsivo  
- Estrutura semântica e acessível  

---

## Como Testar
1. Abra o projeto pelo link acima ou localmente abrindo `index.html` no navegador.  
2. Navegue entre as páginas usando o menu.  
3. Clique em “Começar Agora” para acessar o formulário de cadastro.  
4. Preencha os campos e veja a mensagem de sucesso.
